AWS Infrastructure Monitoring Dashboard

The AWS Infrastructure Monitoring Dashboard is a Laravel-based web application designed to monitor and visualize server health metrics such as CPU utilization, memory consumption, disk usage, uptime, and service status.

This project was developed to gain hands-on experience with infrastructure monitoring, cloud computing concepts, and AWS-inspired architectures commonly used in enterprise environments. The system simulates core monitoring capabilities similar to those provided by Amazon CloudWatch while demonstrating full-stack development and system administration skills.

Key Features
Real-time CPU monitoring
Memory usage tracking
Disk utilization monitoring
Server uptime monitoring
Service health monitoring
Responsive dashboard interface
AWS-inspired architecture documentation
Technologies
